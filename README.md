# Путешествия по России

**Описание проекта**

Учебный проект от Яндекс.Практикум о путешествии по России.
Одностраничный сайт с адаптивным дизайном.

**Технологии используемые в проекте:**
* _Flexbox_
* _БЭМ (Neasted)_
* _Grid layout_
* _Адаптивная вёрстка_
* _медиа-запросы_
